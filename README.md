# 🍽️ Recipe Recommendation System

A full-stack web application that helps users discover, explore, and manage recipes efficiently.

---

## 🚀 Features

* 🔍 Search and explore recipes
* 📖 View detailed recipe instructions
* ❤️ Save favorite recipes
* 🤖 AI-based recommendations *(if implemented)*
* 🔗 Backend API integration

---

## 🛠️ Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS

### Backend

* Python (FastAPI / Flask)

### Database

* Supabase / PostgreSQL

## ⚙️ Installation
