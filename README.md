# 🍽️ Recipe Recommendation System

A full-stack web application to discover and explore recipes.

---

## 🚀 Features
- Search recipes  
- View detailed instructions  
- Save favorites  
- AI-based suggestions  

---

## 🛠️ Tech Stack
- React + TypeScript  
- Vite  
- Tailwind CSS  
- Python Backend  
- Supabase  

---

## ⚙️ Run Locally
npm install  
npm run dev
